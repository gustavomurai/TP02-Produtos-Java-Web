# 📦 Sistema de Gestão de Produtos - CRUD Java Web

Projeto desenvolvido para a disciplina de **Sistemas Web 1 (CBTSWE1)** do curso de Análise e Desenvolvimento de Sistemas no **IFSP - Campus Cubatão**. O objetivo é gerir uma entidade de produtos utilizando tecnologias Java Servlets e armazenamento em memória.

* **Gustavo Cerqueira Murai** 
* **Igor Murai**

## 🎯 Objetivo do Projeto
Desenvolver uma aplicação Java Web completa (CRUD) que permita:
* Cadastrar produtos.
* Listar produtos cadastrados.
* Editar informações de produtos existentes.
* Eliminar produtos da base de dados.
* **Diferencial:** Página de créditos dedicada à equipa.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java 17+
* **Ambiente:** Eclipse IDE
* **Servidor:** Apache Tomcat 9.0
* **Frontend:** HTML5, CSS3 (Design System Customizado), Bootstrap 5.3 e Bootstrap Icons.
* **Arquitetura:** Java Servlets e Padrão DAO (Banco em Memória).

## 📋 Requisitos da Entidade (Produto)
Conforme especificado no roteiro do trabalho, cada produto possui:
- `id` (Gerado automaticamente)
- `nome` (String)
- `unidadeCompra` (int)
- `descricao` (String)
- `qtdPrevistoMes` (double)
- `precoMaxComprado` (double)

## 🎨 Interface e UX
A aplicação foi desenvolvida com um **Design System Premium**, focado em:
* **Modo Escuro (Dark Mode):** Estética moderna e profissional.
* **Tipografia Inter:** Foco na legibilidade.
* **Responsividade:** Compatível com diferentes tamanhos de ecrã.
* **Feedback Visual:** Telas de sucesso e erro personalizadas para melhor experiência do utilizador.
